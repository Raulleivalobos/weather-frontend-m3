# ClimaCL - App de Pronóstico del Clima ⛅

_(Proyecto Módulo 3 - Bootcamp)_

Aplicación web interactiva para consultar el clima actual y el pronóstico semanal de distintas localidades. En esta iteración se ha refactorizado completamente el diseño aplicando metodologías modernas de desarrollo frontend.

## 🚀 Novedades de esta versión

- **Arquitectura SASS:** Código modularizado en directorios (`base/`, `components/`, `layout/`) utilizando directivas `@use`.
- **Metodología BEM:** Clases estructuradas lógicamente para asegurar la escalabilidad de los componentes (ej: `.place-card__temp`).
- **Tema Oscuro (Dark Theme):** Interfaz rediseñada con tonos pizarra y un degradado atmosférico de "Luz de Luna" para reducir la fatiga visual.
- **Integración Nativa:** Inyección de variables CSS (`:root`) para controlar los colores de Bootstrap 5 sin forzar el código con `!important`.

## 🛠️ Tecnologías y Herramientas

- HTML5 (Semántico)
- CSS3 / SASS (SCSS)
- Bootstrap 5
- JavaScript Vanilla
- Git y GitHub
