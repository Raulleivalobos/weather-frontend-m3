\# Nombre del Proyecto App ClimaCl



> Una aplicación web simple para ver el clima de 10 principales ciudades de de Chile



!\[Vista previa del proyecto](ruta/a/tu/screenshot.png) <!-- Opcional: añade un GIF o imagen -->



\## 🛠️ Tecnologías Utilizadas



\*   \*\*HTML5\*\* - Estructura del contenido.

\*   \*\*CSS3\*\* - Estilos y diseño responsivo.

\*   \*\*JavaScript (ES6)\*\* - Lógica funcional y manipulación del DOM.



\## 🚀 Características



\- \[ ] Añadir nuevas tareas.

\- \[ ] Marcar tareas como completadas.

\- \[ ] Eliminar tareas.

\- \[ ] Almacenamiento local (LocalStorage) para no perder datos.



\## 📦 Cómo empezar (Instalación)



1\.  Clona el repositorio:

&#x20;   ```bash

&#x20;   git clone https://github.com/Raulleivalobos/weather-frontend-m2.git    ```

2\.  Abre el archivo `index.html` en tu navegador favorito.

3\.  O utiliza una extensión como {Link: Live Server en VS Code https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer} para verlo en vivo.



\## 💻 Uso



1\.  Escribe la tarea en el campo de texto.

2\.  Haz clic en "Agregar" o pulsa Enter.

3\.  Haz clic en el círculo al lado de la tarea para completarla.
